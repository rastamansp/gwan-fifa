# public/

Assets estáticos servidos pelo Vite na raiz (`/`).

## `demo-foto.jpg` (necessário para o tutorial — F10)

O modal de demonstração (`DemoModal`) gera uma figurinha de exemplo a partir de
`/demo-foto.jpg`. **Este arquivo não é versionado** (ver `.gitignore`).

Coloque aqui um retrato (rosto) **com direito de uso** — foto própria, de modelo
liberado ou royalty-free. **Nunca** use fotos/figurinhas/logos oficiais de
terceiros (Panini/FIFA/clubes/atletas) — ver `RN-F10-01` / `NFR-IP-01` no SDD.

Sem o arquivo, o tutorial degrada com elegância: o passo 4 exibe um aviso e o
restante da página segue funcionando.
